1. 윈도우에 웹서버 설치
* 아파치(Apache) 설치
1) bitnami wamp stack 검색
2) bitnami 들어가서  WAMP 다운로드 
3) 다운로드 완료 후 manager_window가 설치된 것을 확인
- apache2 폴더 -> htdocs 폴더 확인

2. 웹서버와 http
index.html -> Web server -> Web Brower 이렇게 연결해야 웹브라우저에서 http:// 로 표시된다.
index.html-> Web Brower 이렇게 연결되면 file:// 로 표시된다.

http : Hyper Text Transfer Protocol

3. 웹브라우저와 웹 서버 통신
- 컴퓨터의 인터넷 연결하는 곳에 우클릭 후 네트워크 및 인터넷 설정을 들어간 후
어댑터 옵션 변경 클릭 -> 이더넷 -> 자세히 클릭 -> IPv4 주소 확인
이것이 자신의 컴퓨터 주소이다.
* 다른 컴퓨터에서 자신이 만든 웹페이지를 보여주고 싶다면 같은 와이파이를 쓰고 있어야한다.
* 다른 컴퓨터에 내 컴퓨터 IPv4 주소를 치고 파일명을 쓰면 들어갈 수 있다.
